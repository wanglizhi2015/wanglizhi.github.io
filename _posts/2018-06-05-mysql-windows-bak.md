#### 1、新建mysql_bakup.bat脚本如下：
```
@echo off
echo ------------------------- 
echo mysql backup 
echo ------------------------- 
set "Ymd=%date:~,4%%date:~5,2%%date:~8,2%"
cd C:\Program Files\MySQL\MySQL Server 5.7\bin
mysqldump -h localhost -u root -123456 -P 3306 bhxq_db >C:\db_backup\db_%Ymd%.sql
cd \ 
@echo off 
```
